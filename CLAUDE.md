# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目概述

这是一个多语言隐私政策网站项目，为 Nomii 应用程序提供隐私政策和隐私选择中心。项目包含静态 HTML 文件，支持英语、简体中文、繁体中文、日语和韩语五种语言。

## 项目结构

### 核心文件
- `privacy-policy.html` - 英语版隐私政策（主文件）
- `privacy-choices.html` - 英语版隐私选择中心
- `privacy-policy-[语言代码].html` - 各语言版本的隐私政策
- `privacy-choices-[语言代码].html` - 各语言版本的隐私选择中心

### 语言代码
- `zh-hans` - 简体中文
- `zh-hant` - 繁体中文
- `ja` - 日语
- `ko` - 韩语

## 架构特点

### 静态网站架构
- 纯 HTML/CSS 静态文件，无构建流程或依赖项
- 响应式设计，支持移动设备
- 每个页面都包含语言切换导航

### 样式系统
- 内联 CSS 样式，每个文件自包含
- 使用现代 CSS 属性（flexbox、CSS Grid）
- 一致的设计语言：深色头部、浅色主体、灰色页脚

### 内容组织
- 结构化 HTML 语义标记
- 清晰的章节层次（使用 `<section>` 和标题标签）
- 多语言内容保持结构和样式的一致性

## 开发工作流程

### 添加新语言版本
1. 复制现有的英语文件
2. 翻译内容，保持 HTML 结构不变
3. 更新语言代码和文件名
4. 在所有文件的语言切换部分添加新语言链接

### 更新内容
1. 同时更新所有语言版本的对应内容
2. 保持日期和版本号同步
3. 确保链接和联系方式在所有版本中一致

### 样式修改
1. 需要在每个文件中同步更新 CSS
2. 保持响应式设计
3. 确保跨语言的文本显示效果

## 重要注意事项

- 所有文件都是自包含的，修改时需要同步更新
- 语言切换链接需要保持完整性和一致性
- 联系信息和法律条款在所有语言版本中必须保持一致
- 更新生效日期时需要同步所有文件
