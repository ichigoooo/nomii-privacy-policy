Nomii 개인정보 처리방침

시행일: 2025년 12월 1일

언어: English 简体中文 繁體中文 日本語 한국어

1. 소개

Nomii(이하 “회사”)는 이용자의 개인정보 보호를 중요하게 생각합니다. 본 방침은 Nomii 앱과 관련 서비스 이용 시 수집되는 정보와 그 처리 방법을 설명합니다.

2. 수집하는 정보

사용자가 제공하는 정보

  • 이메일, 전화번호, Apple ID(appAccountToken) 등 로그인 정보
  • 닉네임, 프로필 사진, 기본 언어, 캐릭터 설정 등 프로필 데이터
  • AI 캐릭터 생성에 사용되는 이미지, 음성, 텍스트
  • Apple 등 결제 파트너로부터 제공되는 거래 ID, 구독 상태

자동 수집 정보

  • 기기 모델, OS 버전, 로캘, 충돌 로그 등
  • 세션 이벤트, 기능 사용 내역, 성능 지표 등 이용 데이터
  • 서버 측 구독/지급 로그, RICE 잔액 상태

3. 이용 목적

  • 계정 인증, 부정 사용 방지, 서비스 안정성 확보
  • AI 페르소나 생성, 이미지 저장, 대화 기능 제공
  • 구독 처리 및 RICE 가상 화폐 지급/차감
  • 고객 지원, 문제 해결, 서비스 개선
  • 법령 준수 및 이용약관 집행

4. 정보 공유

  • 인프라/알림 서비스를 제공하는 클라우드 업체
  • 구독을 검증하는 Apple 등 결제 사업자
  • Nomii 기능 제공에 필요한 AI 및 분석 파트너 (필요 최소 범위)
  • 법적 요구가 있는 경우 감독 기관 또는 사법 당국

회사는 개인정보를 제3자에게 판매하지 않습니다. 제3자 광고나 교차 앱 추적에 데이터를 사용하지 않으며, 광고 네트워크에 디바이스 식별자를 공유하지 않습니다.

5. 제3자 서비스

본 애플리케이션은 관련 기능을 제공하기 위해 다음 제3자 서비스를 통합합니다. 사용 과정에서, 이러한 서비스가 귀하의 개인정보를 수집하고 처리할 수 있습니다. 각 서비스의 자세한 설명은 다음과 같습니다:

1. OpenRouter AI 서비스

  • SDK 이름: OpenRouter API
  • 사용 목적: Gemini 2.5 Flash Image (Nano/Banana) AI 이미지 생성 서비스 제공
  • 운영자: OpenRouter Inc.
  • 수집하는 개인정보 유형: API 요청 내용, 디바이스 식별자, 사용 로그
  • 개인정보처리방침 링크: https://openrouter.ai/privacy

2. 화산방패 AI 서비스

  • SDK 이름: 화산방패 API
  • 사용 목적: 더우바오 1.6 Vision (Doubao 1.6 Vision) AI 비전 서비스 제공
  • 운영자: 북경 화산엔진유한공사
  • 수집하는 개인정보 유형: API 요청 내용(이미지 데이터 포함), 디바이스 정보, 사용 로그
  • 개인정보처리방침 링크: https://www.volcengine.com/docs/82379/1104498?lang=zh

3. AWS Bedrock AI 서비스

  • SDK 이름: AWS Bedrock API
  • 사용 목적: DeepSeek V3.1 및 DeepSeek R1 대규모 언어 모델 서비스 제공
  • 운영자: Amazon Web Services, Inc.
  • 수집하는 개인정보 유형: API 요청 내용, 디바이스 식별자, 사용 로그
  • 개인정보처리방침 링크: https://aws.amazon.com/privacy
  • 서비스 약관 링크: https://aws.amazon.com/service-terms

6. 국외 이전

데이터는 미국, 싱가포르 등 여러 국가에서 처리될 수 있으며, 계약적/기술적 보호조치를 적용합니다.

7. 보유 기간

서비스 제공 및 법적 의무를 이행하는 데 필요한 기간 동안만 데이터를 보관합니다. 앱 내에서 계정 삭제를 완료하면 핵심 계정 데이터(프로필, 채팅, 생성 콘텐츠, 아바타, 토큰 등)를 삭제하고 세션을 무효화합니다. 거래 기록은 법적 요구에 따라 일정 기간 보관될 수 있으며, 진단/분석 로그는 보안 및 성능 개선을 위해 제한된 기간 저장합니다.

8. 계정 삭제

앱에서 바로 삭제할 수 있습니다: Profile > 계정 작업 > 계정 삭제. 요청은 즉시 처리되며 이메일이나 외부 웹사이트는 필요 없습니다.

  • 처리 결과: 프로필, 채팅 기록, 생성 미디어, 아바타, 활성 토큰을 삭제하고 로그아웃합니다.
  • Sign in with Apple: 삭제 완료 시 서버에서 연동된 토큰을 철회합니다.
  • 구독 안내: 계정 삭제는 Apple 구독을 자동 취소하지 않습니다. 설정 > Apple ID > 구독에서 관리하세요.
  • 법적 예외: 법적 요구나 사기 방지를 위해 제한된 기록을 보관할 수 있습니다.

9. 이용자 권리

  • 앱 내 설정에서 개인정보를 조회·수정·삭제하거나, 저희에게 연락할 수 있습니다.
  • 프라이버시 선택 센터를 통해 데이터 내보내기 또는 계정 삭제를 요청할 수 있습니다.
  • 마케팅 알림 및 분석 동의 설정을 언제든 변경할 수 있습니다.

10. 아동 보호

Nomii는 각 지역 법령이 정한 최소 연령 이상의 사용자를 대상으로 하며, 아동의 정보를 의도적으로 수집하지 않습니다.

11. 정책 변경

방침 변경 시 "시행일"을 업데이트하고 앱 알림이나 이메일로 안내합니다.
